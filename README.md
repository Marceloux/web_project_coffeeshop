# Triple Espresso

## Descrição do projeto

O projeto **Triple Espresso** é uma página web fictícia de uma cafeteria localizada na biblioteca Triple Peaks.

A página apresenta uma estrutura completa com:

- Cabeçalho com navegação;
- Seção de receitas com vídeos incorporados do YouTube;
- Seção de reserva de mesa com formulário;
- Rodapé com informações de mídia social.

O objetivo do projeto é praticar HTML e CSS avançado, seguindo a metodologia BEM para organização das classes e separando os estilos por blocos em arquivos CSS individuais.

## Tecnologias utilizadas

- HTML5
- CSS3
- Metodologia BEM
- Flexbox
- Formulários HTML
- Iframes
- Normalize.css
- Google Fonts
